Páginas Web responsivas

Como fazer páginas web para qualquer dispositivo.

Para conseguir desenvolver um design responsivo para a web, três tecnologias principais(ou modos de aplicar essas tecnologias, se preferir) estão envolvidas: Layout Fluido; Imagens e recursos flexíveis, e; Media Queries.

Layout Fluido

Para conseguir um layout fluido num projeto web, a principal medida a ser tomada é: Não Usar MEDIDAS ABSOLUTAS!! Você pode usar em e %. Use porcentagem para lidar com tamanhos no layout e ems para lidar com fontes. A fórmula mágica para o web responsivo é : Alvo / Contexto = Resultado. Existe uma certa convenção de que o tamanho padrão de fontes em browser desktop é de 16px. Divida o tamanho da fonte pelo tamanho do contexto, e você vai achar o resultado em ems.
